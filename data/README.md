# 数据文件目录

此目录用于存放用户数据导入文件。

## 文件说明

- `users_example.csv` - 示例CSV文件，展示了正确的数据格式

## 使用说明

1. 将你的用户数据整理成Excel格式（.xlsx）
2. Excel文件第一行应为表头，包含以下列：
   - 工号（必填）
   - 姓名（必填）
   - 部门（可选）
   - 职位（可选）
   - 身份证号（可选，用于自动生成密码）
   - 密码（可选）

3. 执行导入命令：
   ```bash
   npm run import:users data/your_file.xlsx
   ```

## 注意事项

- 确保Excel文件格式正确
- 工号必须唯一
- 如果没有提供密码，系统会使用身份证号后六位作为密码
- 如果既没有密码也没有身份证号，将使用默认密码 '123456'
